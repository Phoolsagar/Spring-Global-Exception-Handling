# 🚀 Spring Global Exception Handling

This repository contains multiple examples of **Global Exception Handling in Spring Boot** using REST APIs.

Each module demonstrates different approaches to handling exceptions in a clean, scalable, and production-ready way.

- Handling Specific Exceptions

- HTTP Status Code Management

- `@ControllerAdvice`
- `ResponseEntity`
